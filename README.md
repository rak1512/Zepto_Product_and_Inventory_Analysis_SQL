# Zepto_Product_and_Inventory_Analysis_SQL
Zepto Product &amp; Inventory Analysis using SQL Analyzed 3.7K+ product records using PostgreSQL to explore pricing, discounts, inventory trends, and revenue insights. Implemented data cleaning, aggregation, window functions, and business logic to generate actionable retail insights.
Key Features:

Data cleaning (handling NULLs, removing invalid pricing)
Category-level revenue and inventory analysis
Discount and pricing comparison
Out-of-stock and stock availability insights
Price-per-gram and value-based product analysis
Identification of overpriced and high-discount products

Tools Used:

PostgreSQL
SQL (Aggregations, Window Functions, CTEs, Conditional Logic)
